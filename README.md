# Netflix Homepage Clone

## Description
This project is a Netflix homepage clone created using HTML, CSS, and JavaScript. It recreates the look and layout of the Netflix landing page with a responsive design. This project helped me improve my frontend development skills by working with layouts, styling, and basic JavaScript.

## Features
- Responsive homepage layout
- Navigation bar
- Hero section
- Email input field
- Sign In button
- Modern UI inspired by Netflix
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
- JavaScript
